# 🎬 Netflix Clone

A fully responsive Netflix landing page clone built using **HTML5** and **CSS3**. This project recreates the look and feel of Netflix's official landing page while focusing on responsive web design, modern layouts, and clean user interface development.

## 🌐 Live Demo

🔗 https://kritika-123-eng.github.io/Netflix-Clone/

---

## 📌 Features

- 🎨 Modern Netflix-inspired UI
- 📱 Fully Responsive Design
- 🖥️ Optimized for Desktop, Tablet, and Mobile devices
- 🎬 Hero Banner Section
- 📺 Feature Sections
- ❓ FAQ Section
- 📩 Email Signup Section
- 👣 Footer with Navigation Links
- ⚡ Smooth layout using Flexbox and CSS Grid

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- Flexbox
- CSS Grid
- Media Queries
